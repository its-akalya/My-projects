Portfolio Website

MindfulMe is a simple and user-friendly personal wellness app designed to help users keep track of their daily emotions and thoughts. The interface allows users to select their current mood from a set of options such as Happy, Sad, Excited, Stressed, Relaxed, or Angry, and then write a short journal entry describing their thoughts for the day. The app provides: ✔ A clean and calming UI for easy focus ✔ Mood selection using radio buttons ✔ A large text area for writing journal notes ✔ A “Save Entry” button to store the day’s mood and thoughts ✔ A “View Past Entries” feature to revisit previously saved moments MindfulMe encourages self-reflection, emotional awareness, and daily journaling habits. It’s a great beginner-friendly Python application that demonstrates skills in GUI design (Tkinter), file handling, and user interaction
